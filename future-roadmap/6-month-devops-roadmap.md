# ⚡ 🔮 6-Month DevOps Roadmap

> Built on top of the 7-week industrial training. This plan takes you from Junior to Mid-level DevOps Engineer.

---

## ✦ 📍 Where You Are Now (After 7 Weeks)

```
✅ Linux (Basic to Advanced)
✅ Networking Fundamentals
✅ Shell Scripting
✅ Git & GitHub
✅ Docker & Docker Swarm
✅ AWS Basics (EC2, VPC, Security Groups)
✅ Jenkins CI/CD (Pipelines, Agents, Webhooks)
```

---

## ✦ 🗓️ Month 1–2: Container Orchestration

**Goal:** Learn Kubernetes — the industry standard for running containers at scale.

| Topic | What to Learn |
|---|---|
| Kubernetes (K8s) Basics | Pods, Nodes, Clusters, kubectl |
| K8s Workloads | Deployments, ReplicaSets, StatefulSets |
| K8s Networking | Services, Ingress, DNS |
| K8s Storage | PersistentVolumes, PersistentVolumeClaims |
| Helm Charts | Package manager for Kubernetes |
| Minikube / Kind | Run Kubernetes locally for practice |

```
Docker → Docker Swarm → Kubernetes
(You already know the first two!)
```

---

## ✦ 🗓️ Month 3: Cloud Deep Dive (AWS)

**Goal:** Get comfortable with AWS services used in real DevOps work.

| Topic | What to Learn |
|---|---|
| IAM | Users, Roles, Policies, least privilege |
| S3 | Object storage, versioning, lifecycle policies |
| RDS | Managed databases |
| Load Balancers | ALB, NLB — distribute traffic |
| Auto Scaling | Scale EC2 automatically |
| CloudWatch | Monitoring and alerts |
| Route 53 | DNS management |
| EKS | Managed Kubernetes on AWS |

**Certification to target:** AWS Certified Cloud Practitioner → AWS Solutions Architect Associate

---

## ✦ 🗓️ Month 4: Infrastructure as Code (IaC)

**Goal:** Stop clicking in the AWS console — manage infrastructure with code.

| Tool | Purpose |
|---|---|
| **Terraform** | Provision any cloud infrastructure with code |
| **Ansible** | Configure servers automatically |

Terraform example workflow:
```
Write .tf files → terraform init → terraform plan → terraform apply
```

---

## ✦ 🗓️ Month 5: Monitoring & Observability

**Goal:** Know when things break before users notice.

| Tool | Purpose |
|---|---|
| **Prometheus** | Collect metrics from your systems |
| **Grafana** | Build dashboards from Prometheus data |
| **ELK Stack** | Elasticsearch + Logstash + Kibana for log management |
| **Alertmanager** | Send alerts to Slack/email when things go wrong |

---

## ✦ 🗓️ Month 6: GitOps & Advanced CI/CD

**Goal:** Implement modern GitOps workflows used in production companies.

| Topic | What to Learn |
|---|---|
| **ArgoCD** | GitOps — deploy Kubernetes apps from Git automatically |
| **GitHub Actions** | CI/CD built into GitHub |
| **Advanced Jenkins** | Shared libraries, multibranch pipelines |
| **Security in CI/CD** | Secrets management, image scanning, SAST |
