# 🔧 Week 03 — Internet, Server Setup & Hardware

> **Duration:** Feb 03 – Feb 09, 2026

## Concepts Learned

### How the Internet Works
1. Type `google.com` → DNS resolves to IP
2. Browser sends HTTP request to that IP
3. Server responds with HTML/CSS/JS
4. Browser renders the page

### HTTP vs HTTPS
- HTTP = plain text (Port 80)
- HTTPS = encrypted with TLS (Port 443)

### Shell Scripting for Automation
Automated with shell scripts:
- Installing multiple PHP versions with different ports
- Installing WordPress
- Configuring Nginx reverse proxy

### Computer Hardware
- RAM — temporary memory, can be upgraded
- HDD vs SSD — mechanical vs solid-state storage
- CMOS battery — keeps BIOS settings when powered off
- ROM — read-only memory, stores firmware
