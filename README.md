## рҹ’Ў Things I Learned the Hard Way

> вҡ пёҸ These are hard lessons вҖ” save yourself the pain!

- Always test your SSH keys **before** adding them to Jenkins credentials
- Docker containers are **not** VMs вҖ” they share the host kernel
- `git rebase` rewrites history вҖ” never rebase a shared/public branch
- Port conflicts in Docker are easy to miss вҖ” always check with `docker ps`
- Cron job timing syntax has **5 fields** вҖ” count them every time
- A misconfigured security group in AWS will silently block all traffic
- `EXPOSE` in Dockerfile is documentation only вҖ” use `-p` to actually publish
